# 🐳 Integração Contínua: Pipeline Docker no GitHub Actions (Alura)

📚 Curso da plataforma Alura focado em integrar Docker com pipelines de CI utilizando GitHub Actions.

* 🔗 Curso na Alura: *https://www.alura.com.br/curso-online-integracao-continua-pipeline-docker-github-actions*

---

## 🎯 Sobre o curso

Este curso tem como objetivo mostrar como integrar o uso de Docker dentro de pipelines de integração contínua com GitHub Actions.

Ao longo das aulas, são trabalhados conceitos como automação de builds, execução de testes e criação de containers de forma automática, organizando melhor o fluxo de desenvolvimento e entrega.
