# 🔄 Integração Contínua: pipelines e testes automatizados com GitHub Actions (Alura)

📚 Curso da plataforma Alura focado em implementar pipelines de integração contínua utilizando GitHub Actions.

---

## 🎯 Sobre o curso

Este curso tem como objetivo mostrar como automatizar processos de build, teste e entrega utilizando GitHub Actions.

Ao longo das aulas, são apresentados conceitos de integração contínua, configuração de pipelines e automação de tarefas, ajudando a tornar o fluxo de desenvolvimento mais confiável e eficiente.
