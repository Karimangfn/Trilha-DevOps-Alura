# 🐳 Docker: criando e gerenciando containers (Alura)

📚 Curso da plataforma Alura focado em introduzir o uso de containers com Docker no desenvolvimento de aplicações.

* 🔗 Curso na Alura: *https://www.alura.com.br/curso-online-docker-criando-gerenciando-containers*

---

## 🎯 Sobre o curso

Este curso tem como objetivo ensinar como utilizar o Docker para criar, configurar e gerenciar containers.

Ao longo das aulas, são apresentados conceitos como criação de imagens, persistência de dados, redes e uso do Docker Compose, permitindo rodar aplicações de forma isolada e mais fácil de configurar em diferentes ambientes.
