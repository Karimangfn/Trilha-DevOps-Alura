---

## 🎯 Sobre a trilha

Esta trilha tem como objetivo ensinar os fundamentos da **cultura DevOps**, incluindo colaboração, automação, monitoramento e integração contínua.

Você aprenderá a usar ferramentas essenciais do mercado, como **Docker, Jenkins, Ansible e Terraform**, além de compreender conceitos como:

* Integração e entrega contínua (CI/CD)
* Infraestrutura como código (IaC)
* Conteinerização e orquestração
* Monitoramento e observabilidade
* Feedback contínuo e melhoria de processos

O foco é transformar o modo como você gerencia sistemas, tornando-os **escaláveis, confiáveis e automatizados**.

---

## 🚀 O que você vai aprender

* Planejar, provisionar e gerenciar infraestrutura como código
* Criar e gerenciar containers com Docker
* Construir pipelines de CI/CD usando GitHub Actions
* Implementar monitoramento e alertas em produção
* Aplicar práticas DevOps no dia a dia de projetos reais
