# ⚙️ Ansible: implementando sua infraestrutura como código (Alura)

* 🔗 Curso na Alura: *https://www.alura.com.br/curso-online-ansible-implementando-sua-infraestrutura-como-codigo*

---

## 🎯 Sobre o curso

Este curso tem como objetivo mostrar como automatizar tarefas de configuração e deploy utilizando o Ansible.

Ao longo das aulas, são trabalhados conceitos como playbooks, variáveis, templates e roles, permitindo organizar melhor a infraestrutura e facilitar a criação de ambientes mais complexos e reutilizáveis.
